# Task Analyzer Frontend

## Setup

1. Install dependencies:
```bash
pnpm install
```

2. Start development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Features

- Add tasks manually or via JSON import
- Multiple analysis modes (Smart Balance, High Impact, etc.)


## Usage

1. Add tasks using the form or bulk import
2. Select analysis mode
3. Click "Analyze Tasks" for prioritization
4. Use "Suggest Today" for sample
